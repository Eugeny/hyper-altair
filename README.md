## Install
Install Altair either:

###### The fastest way
1. `hyper i hyper-altair`
2. Taddaa! Enjoy.

###### The old way
1. Edit `~/.hyper.js`.
2. Add `hyper-altair` to the `plugins` array.
3. Taddaa! Enjoy.
